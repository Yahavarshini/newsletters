## 📁 Folder Structure

```
newsletters/
  2026-05/
    index.html              ← May 2026 newsletter
    whatsapp-calling.html   ← WhatsApp Calling feature detail
  _template/
    newsletter.html         ← Copy this to start a new month
    feature-detail.html     ← Copy this for each new feature
```

## 🚀 How to Add a New Month

1. Copy `_template/newsletter.html` → `2026-06/index.html`
2. Copy `_template/feature-detail.html` → `2026-06/feature-name.html`
3. Find all `✏️ EDIT` comments and update them
4. Commit & push → GitHub Pages auto-publishes it

## ⚙️ GitHub Pages Setup

Go to **Settings → Pages → Source: Deploy from branch `main` / `/ (root)`**

Your URL will be: `https://yahavarshini.github.io/newsletters/`
